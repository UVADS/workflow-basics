# Installation

Installation instructions for workflow automation tools.

## Installing Dependencies

### Ruby and Jekyll

If you're working with Jekyll-based sites:

```bash
gem install bundler
bundle install
```

### Node.js and npm

For Node.js-based workflows:

```bash
npm install
```

## Verification

Verify your installation:

```bash
bundle exec jekyll --version
```
